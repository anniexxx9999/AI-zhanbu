## API 使用

### 端点
`POST /api/chart-analysis`

### 请求示例

```bash
curl -X POST http://localhost:3001/api/chart-analysis \
  -H "Content-Type: application/json" \
  -d '{
    "birthInfo": {
      "name": "张三",
      "date": "1990-05-15",
      "time": "10:30:00",
      "city": "北京",
      "latitude": 39.9042,
      "longitude": 116.4074
    },
    "prompt": "请对我进行全面的吠陀占星分析"
  }'
```

### 响应结构

```json
{
  "success": true,
  "data": {
    "analysis": "一份1500-2500字的专业占星分析报告...",
    "chartData": { /* 完整的星盘数据 */ },
    "metadata": {
      "generatedAt": "2024-01-01T00:00:00.000Z",
      "source": "deepseek-ai",
      "prompt": "用户提示词"
    }
  }
}
```
