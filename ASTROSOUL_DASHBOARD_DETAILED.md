## 4. D9 Navamsa婚姻盘

### 4.1 D9盘简介

Navamsa (D9) 是印度占星中最重要的分盘之一，被称为"命运之盘"或"婚姻之盘"。它将每个星座的30度分成9个部分（每部分3°20'），用于查看：

1. **婚姻与配偶**: 配偶的性格、外貌、相遇方式
2. **命运转折**: 人生下半场的主题
3. **灵性成长**: 内在的真实自我
4. **行星力量**: 验证D1盘中行星的真实力量

### 4.2 D9盘计算示例

以上升双子座13°43'为例：

```
双子座13°43' 在 D1盘
↓
计算在D9盘中的位置：
13°43' ÷ 3°20' = 4.11 → 落在第5个Navamsa区间
双子座是奇数星座，从本星座开始计数
第5个Navamsa = 天秤座

所以D9盘上升 = 天秤座
```
