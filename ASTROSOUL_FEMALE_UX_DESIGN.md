### 1.2 情感化设计原则

```
原则 1: "像闺蜜，不像专家"
  → 语言温暖、共情、鼓励
  → 用"你"而非"用户"
  → 用emoji和感叹号表达情绪

原则 2: "美到想截图"
  → 每一屏都可以是壁纸
  → 配色柔和、渐变丰富
  → 细节精致、动画流畅

原则 3: "必须晒出去"
  → 生成精美的分享卡片
  → 鼓励社交互动
  → 制造"仪式感"时刻

原则 4: "让我感觉特别"
  → 个性化称呼和内容
  → 提供定制选项
  → 庆祝每一个里程碑
```

## 2. 色彩与氛围重塑

### 2.1 现有配色的问题

```
当前配色:
  主色: 午夜靛蓝 (#191D3A) - 太深、太冷、太"理工男"
  强调色: 天体金 (#E3B37C) - 还可以，但不够温暖
```

### 3.3 字体优化

```css
/* 重新设计的字体层级 */

/* 标题 - 更柔美的衬线字体 */
--font-display: 'Playfair Display', serif;  /* 替代Lora */
/* 特点: 更优雅、更女性化、更editorial */

/* 正文 - 保持Inter，但调整字重 */
--font-body: 'Inter', sans-serif;
/* 使用更细的字重: 400而非500 */

/* 装饰性文字 - 手写风格 */
--font-script: 'Dancing Script', cursive;
/* 用于: 引言、特殊强调、签名 */

/* 示例应用 */
.page-title {
  font-family: var(--font-display);
  font-size: 56px;
  font-weight: 400;  /* 不用700 Bold */
  letter-spacing: -0.5px;
  color: transparent;
  background: var(--accent-primary);
  -webkit-background-clip: text;
  background-clip: text;
}

.ai-quote {
  font-family: var(--font-script);
  font-size: 24px;
  color: var(--accent-secondary);
  font-style: italic;
}

.body-text {
  font-family: var(--font-body);
  font-size: 17px;  /* 从16px增大 */
  font-weight: 400;  /* 从500减轻 */
  line-height: 1.9;  /* 从1.8增加 */
  letter-spacing: 0.3px;  /* 增加字间距，更舒适 */
}
```
